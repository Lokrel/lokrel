lokrel

A quiet macOS library for 3D printing models, previews, metadata, source links, tags, and notes.

用照片管理软件的方式整理 3D 打印模型。把同一模型的 3MF、STL、STEP、OBJ、图片、PDF 和说明文件合并成一个清晰项目。

Download for macOS View Source

macOS 14 or later. Public releases are distributed through GitHub while signing and notarization are being finalized.

Model libraries without spreadsheet energy.

lokrel keeps model files close to their context: rendered previews, embedded 3MF metadata, source links, notes, favorites, and tags.

Project grouping

Matching model, image, PDF, text, and README files in the same folder are shown as one project.

Preview first

Embedded 3MF thumbnails are used when available. STL and OBJ files can render card previews and interactive views.

Clean metadata

Author, license, descriptions, and MakerWorld source links are extracted from messy embedded 3MF metadata.

Local only

Original model files stay untouched. Notes, tags, favorites, and cover choices live in a local database.

Finder-friendly

Open source files in their default apps or reveal the original folder when you need to rename or reorganize files.

Fast search

Find projects by display name, author, tags, or associated filenames as your library grows.

Built around the way model folders actually look.

A three-column workspace keeps folder structure, model cards, and inspector details visible at the same time.
